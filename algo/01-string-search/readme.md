## Boyer-Moore String Search

字符串搜索算法是最常用到一个算法； Naive 的搜索算法的算法复杂度是 O(mn).

目前性能最好的算法是： Boyer-Moore 算法。

### Boyer-Moore 算法

BM 算法主要使用两个 Heuristic Method 来进行改进。

- Bad Character Heuristic Method
- Good Suffix Heuristic Method

BM 算法是需要预处理的。
