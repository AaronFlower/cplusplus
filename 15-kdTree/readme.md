## kdTree

Using KD-Tree to implement kNN algorithm.

- `point.h` defines a k-Dimension Point class.
- `boundedPQueue.h` implements a priority queue with bounded limit.
- `kdTree.h` implement a KDTree.

### How to Use

#### test

```bash
$ cd cloned/path
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./15-kdTree
```
